<h1>Hydraverse</h1>
<p>Sitio web de mi equipo del NASA Space Apps Challenge para mostrar nuestros proyectos, logros y experiencia. Hecho con HTML, CSS, JavaScript y JSON.</p>

<h3>Demo:</h3>
<p>https://hydraverse.vercel.app/</p>

<h3>Tecnologías Utilizadas:</h3>
<ul>HTML</ul>
<ul>CSS</ul>
<ul>JavaScript</ul>
<ul>JSON</ul>

<h3>Estructura del Proyecto:</h3>
<ul>A: contiene imagenes.</ul>
<ul>Archivo HTML</ul>
<ul>Archivo CSS</ul>
<ul>Archivo JavaScript</ul>
<ul>JSON</ul>

<h3>Funcionalidades:</h3>
<ul>Navegación entre secciones de la página con lógica implementada gracias a JavaScript.</ul>
<ul>Listado de Proyectos</ul>
<ul>Sección de "About"</ul>
<ul>Footer</ul>
<ul>Adaptable según el tamaño de pantalla</ul>

<h3>Objetivo del Proyecto:</h3>
<p>Presentar en un solo lugar nuestra experiencia, trabajos, logros y al equipo del Desafío Internacional de las Aplicaciones Espaciales de la NASA.</p>
